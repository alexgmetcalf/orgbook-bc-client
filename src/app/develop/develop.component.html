OrgBook API

Are you developing an application geared towards registered organizations in BC?

OrgBook BC has a REST API (the OrgBook API), allowing developers to integrate publicly available, verifiable data about registered BC organizations into third-party applications.

Documentation for the OrgBook API can be found in the official GitHub repository. It contains a series of guides for implementing the most commonly used features of OrgBook, such as:

  • An official BC organization name search with autocomplete
  • Retrieving and auto-populating form fields with organization info such as registration number, business number, entity type, entity status, etc.
  • Searching for credential issuers
  • Searching for credential types
  • Verifying organization credentials

Feel free to check out the documentation, and contribute to them if you'd like.
